# 동아 인테리어 자재 선택 기록 작업 지침

## 목적

- 이 문서는 인테리어 재시작 후 도배지, 바닥장판, 문, 몰딩, 타일, 위생도기, 수전 등 자재 후보를 하나씩 기록하고 비교하기 위한 기준이다.
- 실제 판단에 쓰는 자료이므로 공간명, 자재명, 색감, 마감, 금액 단위를 특히 조심한다.

## 기준 파일

- HTML 기준 파일: `donga-interior/material_selection.html`
- 이 작업의 데이터 원본은 HTML 안의 `<script type="application/json" id="materialRecordsJson">` 블록이다.
- 화면 표를 직접 고치지 않고 JSON을 먼저 수정한 뒤 렌더링 결과를 확인한다.

## 데이터 구조

- 최상위 구조:

```json
{
  "meta": {
    "project": "동아 인테리어 도배지 선택",
    "updatedDate": "YYYY-MM-DD",
    "amountUnit": "원",
    "statusValues": ["검토중", "보류", "확정"]
  },
  "records": []
}
```

- `records` 항목 구조:

```json
{
  "space": "전체",
  "material": "도배지",
  "candidate": "개나리 로아스",
  "colorTone": "천정/벽면 구분",
  "finish": "천정 87476-1 / 벽면 87476-2",
  "status": "검토중",
  "expectedWon": 0,
  "memo": "천정과 벽면 품번 구분해서 주문 확인"
}
```

## 수정 규칙

- 새 자재 후보를 추가할 때는 `records` 배열에 항목을 추가한다.
- 자재명은 `도배지`, `바닥장판`, `방문`, `시트지`, `문선`, `문손잡이`, `벽타일`, `바닥타일`, `현관타일`, `베란다 타일`, `양변기`, `세면대`, `욕조`, `슬라이드장`, `젠다이`, `수전`, `욕조수전`, `탄성코트`, `스위치`, `걸레받이, 천정 마이너스 몰딩`처럼 사용자가 바로 알아볼 수 있게 적고, 위치는 `space`에 적는다.
- 금액은 모두 `원` 단위 숫자로만 기록한다. 아직 모르면 `0`으로 둔다.
- 상태값은 `검토중`, `보류`, `확정` 중 하나를 사용한다.
- 날짜는 `YYYY-MM-DD` 형식으로 기록한다.
- 제품명이나 브랜드가 확정되면 `candidate`에 정확한 모델명 또는 매장 표기명을 적는다.

## 화면 구조

- 상단: 문서 제목, 수정일, 인쇄 버튼
- 요약: 전체 자재 수, 검토중 수, 확정 수, 예상 합계
- 표: 위치, 구분, 브랜드/제품명, 색감, 규격/마감, 상태, 예상가, 메모
- 하단: 현재 JSON 데이터 확인 영역

## 인쇄 규칙

- 인쇄 시 JSON 확인 영역은 숨긴다.
- 자재 후보 표와 요약은 인쇄에 남긴다.
