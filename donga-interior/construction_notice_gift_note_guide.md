# 인테리어 공사 사전 양해 편지 인쇄물 작업 지침

## 목적

- 동아아파트 28동 901호 인테리어 공사 전에 이웃집 문앞에 걸어둘 작은 편지를 A4 한 장에 여러 장 인쇄할 수 있게 준비한다.
- 편지는 손뜨개 복주머니와 수세미에 함께 걸 수 있는 작은 흑백 문안으로 사용한다.

## 기준 파일

- HTML 기준 파일: `donga-interior/construction_notice_gift_note.html`
- PDF 산출 파일: `donga-interior/construction_notice_gift_note.pdf`

## 문안

```text
안녕하세요, 28동 901호입니다.

집 내부 인테리어 공사로 인해
한동안 소음과 불편을 드릴 수 있어
미리 양해 부탁드립니다.

작은 마음으로 직접 뜬 복주머니와 수세미를 준비했습니다.
너그러운 이해에 감사드립니다.

28동 901호 드림
```

## 인쇄 규칙

- A4 세로 한 장 기준으로 3열 x 3행, 총 9개 편지를 배치한다.
- 흑백 인쇄에 적합하도록 검정 텍스트만 사용하고, 편지별 테두리는 넣지 않는다.
- 자를 때는 편지 사이의 좁은 여백을 기준으로 구분한다.
- 브라우저에서 HTML을 열고 인쇄하거나, HTML을 PDF로 변환해 사용한다.

## 수정 규칙

- 문안을 바꿀 때는 HTML의 `message` 배열과 이 지침의 문안을 함께 확인한다.
- A4 한 장에 들어가는 수량을 바꿀 때는 `.sheet`의 grid 행/열과 스크립트 반복 수를 함께 맞춘다.
