# 현재 수준 평가표 안내

## 목적

- 이 평가는 황소수학 입학시험 준비를 시작하기 전에 현재 수준을 확인하기 위한 1차 진단 시험지다.
- 단순 점수보다 `어떤 유형에서 막히는지`, `HBI-3/HBI-4 구간을 얼마나 버티는지`를 확인하는 것이 목적이다.

## 기준 파일

- 인쇄용 시험지 HTML: `current_level_assessment.html`
- 정답/채점표 HTML: `current_level_assessment_answer.html`
- HBI 판정표 HTML: `current_level_hbi_review.html`

## 사용 방법

- 실제 시험처럼 시간을 재고 풀어 본다.
- 권장 시간은 `70분`이다.
- 풀고 나서 바로 채점만 하지 말고 아래 항목을 함께 기록한다.
  - 막힌 단원
  - 시간이 오래 걸린 문항
  - 계산 실수인지 조건 해석 문제인지
  - 표현 전환이 필요한 문제에서 멈췄는지

## 구성 원칙

- 범위는 `초등학교 4학년 1학기` 기준으로 잡는다.
- 문제는 학교 기본형이 아니라 황소 대비 진단용으로 구성한다.
- 난도는 `HBI-2`부터 `HBI-4`까지 섞되, 중심은 `HBI-3~HBI-4`로 둔다.
