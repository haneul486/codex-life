# 실전판 1 안내

## 목적

- 이 세트는 `현재 수준 평가표`보다 한 단계 더 실전형으로 구성한 황소수학 대비 세트다.
- 범위는 `초등학교 4학년 1학기`를 유지하지만, 문제 구조와 시간 압박은 더 황소답게 설계한다.
- 내일 이 세트를 푼 뒤 `현재 수준 평가 결과`와 함께 비교해서 앞으로의 진도와 약점 보정 우선순위를 정한다.

## 기준 파일

- 인쇄용 실전판 HTML: `real_set_1.html`

## 사용 방법

- 권장 시간은 `80분`이다.
- 문제를 읽고 바로 풀지 말고, 먼저 `쉬운 문제 / 보류할 문제`를 빠르게 나눈다.
- 한 문제를 오래 잡지 말고, 막히면 표시하고 넘어간 뒤 다시 돌아온다.
- 풀이가 끝난 뒤 아래를 함께 기록한다.
  - 가장 오래 걸린 문항
  - 끝까지 못 푼 문항
  - 다시 보니 풀린 문항
  - 계산 실수보다 `조건 해석`, `표현 전환`, `시간 운영` 중 무엇이 더 문제였는지

## 난도 기준

- 중심 난도는 `HBI-4`다.
- 일부 문항은 `HBI-5`로 넣어 상위권 압박과 선별 운영을 확인한다.
- 계산만 어려운 문제가 아니라, `문항 요구 분리`, `표현 전환`, `낯선 구조 해석`이 핵심이 되게 구성한다.
