# kids-eye-care-plan

아이들의 시력 변화, 드림렌즈 결정, 예비 안경 구매, 병원 검진 일정, 생활 루틴, 위험 신호를 한 곳에서 관리하기 위한 가족 프로젝트입니다.

기준 화면은 `index.html`입니다. 화면은 정적 HTML이지만, 주요 내용은 HTML 하단의 JavaScript 데이터 배열을 기준으로 렌더링합니다.

주요 데이터 구조:

- `children`: 김범준·김서연의 현재 시력, 처방전 도수, 다음 확인값, 관찰 신호
- `milestones`: 1년 관리 마일스톤과 완료/진행 필요/미진행 상태
- `actions`: 이번 달 병원·안경원·집 루틴 실행 항목
- `screenPlans`: 온라인 학습·게임·PC 사용 시간분배 기준
- `readinessItems`: 시력관리 준비도 체크 항목
- `examRecords`: 검사 기록 누적표
- `precisionChecklist`: 정밀검진에서 받아올 검사값과 질문 문장
- `decisionRules`: 드림렌즈, 일반 예비안경, 근시억제 안경, 저농도 아트로핀 선택 기준
- `treatmentRecheck`: 의사 조언을 반영한 드림렌즈 vs 일반안경 재판정
- `dangerSignals`: 즉시 착용 중단 또는 병원 문의가 필요한 위험 신호
- `clinicQuestions`, `opticalQuestions`: 병원·안경원 질문지
- `routines`: 생활 루틴 점검 항목
- `monthlyReviews`: 월간 리뷰 보드
- `lensProtocol`: 드림렌즈 시작 후 밤·아침 프로토콜
- `purchases`: 구매·비용 관리표
- `costItems`: 드림렌즈, 일반안경, 관리용품, 근시억제 안경, 아트로핀 예상 비용
- `budgetScenarios`: 1년 예산 시나리오
- `reviewLog`: 부모 관점 자체 검토 및 보완 로그

수정할 때는 화면의 표 마크업을 직접 고치기보다 위 데이터 배열을 먼저 수정합니다.

화면에서 직접 쓰는 기능:

- `오늘의 관찰 기록`: 날짜, 아이, 루틴, 메모를 입력하면 브라우저 `localStorage`에 자동 저장됩니다.
- `위험 신호`: 체크한 항목에 따라 착용 중단/관찰 강화/유지 기준이 즉시 표시됩니다.
- `다음 진료 준비 요약`: 아직 확인해야 할 검사값과 최근 메모를 한 번에 보여줍니다.
- `우리 집 시력관리 준비도`: 마일스톤, 검사값, 생활 루틴, 이번 달 액션을 기준으로 상단 관리점수를 보여줍니다.
- `시력 변화 추세`: 2025년까지의 기준 시력과 2026년 6월 기준 시력을 그래프로 보여줍니다.
- `시력관리 준비도 체크`: 예약, 예비안경, 안축장 질문, 위험 대응, 비용 관리 준비 상태를 저장합니다.
- `온라인 학습·게임·PC 사용 시간분배`: 25분 학습+5분 휴식, 평일/주말 게임 시간, 하루 PC 총량, 야외활동, 화면 거리와 취침 전 화면 기준을 관리합니다.
- `진료실 30초 브리핑`: 의사에게 보여줄 아이별 시력 변화, 처방전 도수, 필수 확인값, 최근 메모를 요약합니다.
- `빠른 이동`: 모바일에서도 핵심 지표, 아이별 카드, 오늘 기록, 위험 신호, 진료 준비로 바로 이동합니다.
- `관리 데이터 내보내기`: 현재 기준 데이터와 브라우저에 저장된 체크·메모 상태를 JSON으로 내려받습니다.
- `저장 상태 초기화`: 브라우저에 저장된 체크와 메모만 초기화합니다. 기준 데이터는 HTML 안에 그대로 남습니다.
- `구매·비용 관리`: 구매비, 관리비, 조건부 비용을 나눠 보고 1년 예산 시나리오를 비교합니다.

## 업데이트 절차

새 검사나 구매가 생기면 아래 순서로 반영합니다.

1. `children`에서 아이별 최신 시력, 도수, PD, 다음 확인값을 갱신합니다.
2. `examRecords`에 검사 날짜, 병원명, 나안시력, 교정시력, 도수, 안축장, 각막 상태를 추가합니다.
3. `milestones`에서 완료된 단계는 `status: "done"`으로 바꾸고, 다음 행동은 `status: "doing"`으로 올립니다.
4. 구매가 있으면 `purchases`에 금액, 구매처, 보증·교체 조건, 다음 교체 예상일을 남깁니다.
5. 드림렌즈를 시작하면 `lensProtocol`과 `dangerSignals`를 실제 병원 안내에 맞게 보완합니다.
6. 월말에는 `monthlyReviews`에 생활 루틴과 증상 메모를 정리합니다.
7. 브라우저에 저장된 메모를 백업하려면 화면 상단의 `관리 데이터 내보내기`를 사용합니다.

## 2. 기본 방향

1. 시력관리와 진행 추적을 메인 목표로 둡니다.
2. 드림렌즈는 메인 선택지로 검토하지만, 결정 기준은 안과 검사값입니다.
3. 드림렌즈를 시작하더라도 예비 안경은 준비합니다.
4. 예비 안경은 근시억제 안경이 아니라 일반안경을 우선합니다.
5. 드림렌즈가 맞지 않거나 관리가 어렵다면 근시억제 안경을 메인 안경으로 전환 검토합니다.
6. 3-6개월 단위로 검사 결과를 보고 판단을 업데이트합니다.

## 3. 선택 기준

의사가 “눈이 더 나빠지지 않게 관리하는 것이 중요하다”고 말한 경우, 일반안경과 드림렌즈의 역할을 분리해서 봅니다.

- 일반안경: 잘 보이게 하는 기본 교정, 학교생활과 학습 시야 확보에 중요
- 일반 예비안경: 드림렌즈를 못 끼는 날 쓰는 안전장치
- 드림렌즈: 각막과 생활 루틴이 맞으면 근시 진행 관리 후보
- 근시억제 안경: 드림렌즈를 하지 않거나 중단해 안경이 메인이 될 때 검토
- 저농도 아트로핀: 3-6개월 추적에서 진행이 빠르면 안과 상담

따라서 “그냥 드림렌즈를 끼면 된다”가 아니라, `각막 지형도`, `안축장`, `교정시력`, `아이 협조`, `위생 루틴`, `수면 시간`을 보고 결정합니다.

현재 가족 기준으로는 김서연이 진행 관리 우선순위가 더 높고, 김범준도 드림렌즈 후보입니다. 두 아이 모두 일반 예비안경은 준비하되, 드림렌즈 시작 여부는 안과 적합 판정 후 결정합니다.

### 드림렌즈를 선택하기 좋은 경우

- 부모가 매일 세척, 보존, 착용 시간을 확인할 수 있음
- 아이가 손 위생과 렌즈 착용 루틴을 따라올 수 있음
- 수면 시간이 비교적 일정함
- 운동이나 활동 때문에 낮에 안경 없는 생활이 유리함
- 안과에서 각막 지형도와 눈 상태가 적합하다고 판단함

### 드림렌즈를 조심해야 하는 경우

- 렌즈 세척과 보관을 꾸준히 관리하기 어려움
- 충혈, 알레르기, 눈 비빔이 잦음
- 수면 시간이 짧거나 불규칙함
- 아이가 렌즈 착용에 강하게 거부감을 보임

## 4. 예비 안경 원칙

드림렌즈가 메인이라면 예비 안경은 `일반안경`으로 충분합니다.

예비 안경의 목적은 근시 진행 억제가 아니라, 다음 상황에서 학교생활과 학습 시야를 지키는 것입니다.

- 드림렌즈를 못 낀 날
- 눈 충혈, 결막염, 통증으로 렌즈를 쉬어야 하는 날
- 렌즈 분실 또는 파손
- 수면 시간이 짧아 낮 시력이 충분히 나오지 않는 날
- 드림렌즈 적응 초기의 시력 변동

근시억제 안경은 매일 오래 착용할 때 의미가 큽니다. 예비용으로 가끔 쓰는 안경이라면 비용 대비 효율이 낮습니다.

### 예상 비용 기준

실제 금액은 병원, 지역, 렌즈 종류, 재피팅 조건에 따라 달라집니다. 견적 전까지는 아래 범위를 예산 기준으로 둡니다.

| 항목 | 예상 비용 | 성격 |
|---|---:|---|
| 일반 예비안경 | 1인 8-20만원 | 구매비 |
| 드림렌즈 | 1인 80-150만원 | 구매비 |
| 드림렌즈 관리용품 | 월 1-3만원 | 관리비 |
| 근시억제 안경 | 1인 30-70만원 | 조건부 구매비 |
| 저농도 아트로핀 | 월 2-5만원 | 조건부 관리비 |
| 정밀검사·피팅 | 5-15만원 이상 | 초기비 |

1년 예산을 볼 때는 `렌즈 구매비`뿐 아니라 `예비안경`, `초기검사`, `관리용품`, `분실·파손·재피팅 조건`을 함께 봅니다.

## 6-1. 정밀검진에서 받아올 것

정밀검진의 목적은 오늘 안경을 맞추는 것에서 끝나지 않습니다. 앞으로 3개월, 6개월, 1년 뒤 근시가 얼마나 진행됐는지 비교할 `기준값`을 만드는 것입니다.

병원에서 아래 항목을 김범준·김서연 각각 받아옵니다.

| 항목 | 왜 필요한가 | 기록 방법 |
|---|---|---|
| 나안시력 | 안경 없이 실제로 얼마나 보이는지 확인 | 우안/좌안 각각 기록 |
| 교정시력 | 도수를 맞췄을 때 1.0까지 나오는지 확인 | 교정 후 우안/좌안 각각 기록 |
| 굴절검사 도수 | 근시·난시 정도를 숫자로 비교 | SPH, CYL, AXIS를 양쪽 눈 각각 기록 |
| 조절마비 굴절검사 여부 | 아이의 조절 힘 때문에 도수가 흔들리는지 확인 | 실시 여부와 결과 도수 기록 |
| 안축장 | 근시 진행 추적의 핵심 숫자 | mm 단위로 우안/좌안 각각 기록 |
| 각막 지형도 | 드림렌즈 적합성 판단 | 적합/주의/부적합과 이유 기록 |
| 눈 건강 상태 | 렌즈 착용 가능성과 중단 기준 판단 | 각막, 결막, 알레르기, 건조 여부 기록 |
| 치료 강화 기준 | 3개월 뒤 어떤 변화면 치료를 바꿀지 결정 | 도수 변화, 안축장 변화 기준 기록 |

병원에서 그대로 말할 문장:

> 선생님, 단순히 안경을 맞추는 것보다 앞으로 근시가 더 진행되는지 추적 관리하려고 합니다. 오늘 기준값으로 남겨야 할 나안시력, 교정시력, 굴절검사 도수, 안축장, 각막 지형도 결과를 아이별로 알려주세요. 3개월 뒤 어떤 수치가 나오면 치료를 강화해야 하는지도 같이 알고 싶습니다.

## 7. 병원에서 확인할 질문

- 아이별 정확한 근시 도수와 난시는 얼마인가요?

- 안축장 길이는 얼마이고, 앞으로 추적 측정이 가능한가요?

- 일반안경만 착용할 경우 근시 진행 억제 관점에서 부족한 부분은 무엇인가요?

## 10. 생활 관리 원칙

- 렌즈를 만지기 전 손을 씻습니다.
- 드림렌즈는 수돗물과 접촉하지 않게 합니다.
- 충혈, 통증, 눈곱, 심한 이물감이 있으면 렌즈 착용을 중단하고 병원에 문의합니다.
- 책과 화면을 너무 가까이 보지 않게 합니다.
- 45분 이상 근거리 작업이 이어지면 잠깐 먼 곳을 보게 합니다.
- 온라인 학습은 25분 단위로 끊고 5분은 먼 곳 보기, 스트레칭, 물 마시기처럼 눈과 몸을 쉬게 합니다.
- 게임은 통제보다 합의와 자율 종료를 원칙으로 합니다. 평일은 30-45분, 주말·방학은 60-90분 정도를 기본 범위로 두고 가족 일정과 야외활동에 맞춰 조정합니다.
- 게임을 시작하기 전에 종료 시간을 정하고, 10분 전 알림 뒤 아이가 스스로 마무리하도록 합니다.
- 게임도 30분마다 잠깐 멈춰 먼 곳 보기, 물 마시기, 몸 움직이기를 넣습니다.
- PC 화면은 50-60cm 이상, 책과 태블릿은 30cm 이상 거리를 둡니다.
- 하루 야외활동은 최소 60분, 가능하면 80-120분을 목표로 합니다.
- 취침 전 1시간은 PC·태블릿·폰 학습을 피합니다.
- 3-6개월마다 도수와 안축장을 확인합니다.

## 11. 현재 결론

현재 가족 의사결정은 다음과 같습니다.

> 드림렌즈를 메인으로 검토한다.  
> 드림렌즈를 시작한다면 예비 안경은 일반안경으로 준비한다.  
> 드림렌즈가 어렵거나 중단되면 근시억제 안경을 메인 안경으로 전환 검토한다.
